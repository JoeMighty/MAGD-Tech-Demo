Motion Design Expressions

Essential After Effects expressions for dynamic motion graphics and animation workflows

Time-Based

Use the time variable for continuous motion effects

Randomization

wiggle() and seedRandom() for organic movement

Seamless Loops

loopOut() variations for perfect cycling animations

01 Rotation
time * 45
Continuous Spin: Rotates the object 45° per second for smooth rotation
Apply to: Rotation property
02 Position/Rotation
wiggle(2, 30)
Random Jitter: Adds organic motion - 2 wiggles per second, 30px/degree range
Apply to: Position, Rotation, or Scale
03 Any Property
loopOut("cycle")
Seamless Loop: Loops the last keyframes in a perfect cycle
Apply to: Any property with 2+ keyframes
04 Any Property
loopOut("pingpong")
Ping Pong: Moves back and forth between keyframes like a yo-yo
Apply to: Any property with keyframes
05 Opacity
Math.sin(time * 2) * 50 + 50
Smooth Fade: Creates a gentle fade in/out loop between 0-100%
Apply to: Opacity property
06 Scale
s = Math.sin(time * 3) * 20 + 100; [s, s]
Breathing Effect: Makes object pulse between 80-120% scale
Apply to: Scale property
07 Position
y = Math.sin(time * 2) * 10; value + [0, y]
Floating Motion: Adds gentle up and down floating movement
Apply to: Position property
08 Any Property
valueAtTime(time - index * 0.2)
Sequence Delay: Delays animation for each layer in sequence
Apply to: Any animated property
09 Position
seedRandom(index, true); wiggle(1, 20)
Unique Wiggle: Each layer wiggles differently - perfect for motion systems
Apply to: Position property
10 Opacity
Math.floor(time % 1) == 0 ? 100 : 0
Blinking Effect: Creates on/off blinking once per second
Apply to: Opacity property
11 Opacity
linear(time % 2, 0, 2, 0, 100)
Linear Fade: Fades from 0 to 100 over 2 seconds, then resets
Apply to: Opacity property
12 Rotation
loopDuration = 2; t = time % loopDuration; t * 180
Smooth Rotation Loop: Rotates 0°-360° in 2 seconds and loops seamlessly
Apply to: Rotation property
13 Color
color1 = [1,0,0,1]; color2 = [0,0,1,1]; t = (Math.sin(time*2)+1)/2; linear(t,0,1,color1,color2)
Color Transition: Smooth transition between red and blue colors
Apply to: Fill Color or Color Control
14 Position
freq = 3; amp = 25; value + [Math.sin(time*freq)*amp, 0]
Side-to-Side Swing: Creates swinging motion like a hanging sign
Apply to: Position property
15 Position
vel = velocityAtTime(time - thisComp.frameDuration); amp = 0.05; freq = 4; decay = 2; vel * amp * Math.sin(freq*time*2*Math.PI)/Math.exp(decay*time)
Bounce Physics: Adds realistic bounce after movement stops
Apply to: Position (after keyframes)
16 Any Property
timeToFrames(time) % 2 == 0 ? value : value + 10
Frame-by-Frame Toggle: Alternates value every other frame for choppy effect
Apply to: Position or Opacity
17 Any Property
thisComp.layer("Null 1").transform.position
Layer Link: Links property to another layer's property
Apply to: Any transform property
18 Opacity
ease(time % 2, 0, 2, 0, 100)
Smooth Ease: Smoothly eases opacity from 0 to 100 every 2 seconds
Apply to: Opacity property
19 Position
value + wiggle(1,10) - value
Subtle Shake: Creates shake around current position without drift
Apply to: Position property
20 Any Property
loopOut("offset")
Offset Loop: Repeats movement, adding offset each cycle (continuous scrolling)
Apply to: Position, Rotation, or Scale

How to Use These Expressions

1

Select Property

In After Effects, select the property you want to animate (Position, Rotation, Scale, etc.)

2

Add Expression

Alt+Click (Windows) or Option+Click (Mac) on the stopwatch icon to enable expressions

3

Paste & Modify

Click any expression above to copy it, then paste and adjust values to fit your needs

4

Preview

Press spacebar to preview your animation and fine-tune the expression parameters
